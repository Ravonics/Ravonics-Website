## BATCH UPDATE INSTRUCTIONS

### Method 1: Manual Copy-Paste (Recommended for accuracy)
1. Open each HTML file
2. Locate the `<head>` section
3. Replace existing meta tags with template above
4. Update page-specific paths in canonical URL
5. Ensure all social media tags use correct page URL

### Method 2: Find & Replace (Faster, requires validation)
1. Create backup of entire website folder
2. Use editor's batch find/replace across multiple files
3. Replace generic meta descriptions with page-specific ones
4. Add missing OpenGraph and Twitter tags
5. Test all pages in browser to verify

### Method 3: Script-Based (For developers)
Create a Node.js script to:
1. Read HTML files

4. Update canonical URLs dynamically
5. Preserve existing custom tags
